== Description ==
Save time to your visitors by putting EKONT shipping module. The Plugin contains all the offices of the company EKONT in Bulgaria.Спестете време на вашите клиенти, интегрирайки този модул за бърза поръчка през ЕКОНТ. Приложението позволява бързо и лесно да бъде избран град и офис на ЕКОНТ или доставка до адрес.

== Installation ==
1. Download the plugin
1. Upload the book-reviews folder to your site's
wp-content/plugins directory
1. Activate the plugin in the WordPress Admin Panel
1. Enjoy
